# Budget-Manager--JAVA
It doesn't sound very serious, but it is very important to manage your budget. This means analyzing your expenses and estimating income, which can sometimes be difficult to do on your own. Fortunately, the technology is there to help: I created your own personal budget manager software that calculates all the extra details and helps control finances.
